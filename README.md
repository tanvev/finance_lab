# Finance Lab 

Finance Lab is an interactive, web-based learning platform designed to simplify **Indian personal finance concepts** through practical tools and simulations. Instead of learning finance only through theory, users can experiment with real-world scenarios using calculators and planners.

**Live Project:** [https://finance-lab.onrender.com/](https://finance-lab.onrender.com/)

---

## Features

* **Budget Planner**

  * Create and manage monthly budgets
  * Visual breakdown of expenses
  * Encourages practical money management habits

* **Tax Calculator (India)**

  * Income tax calculation based on Indian tax rules
  * Comparison of different tax scenarios
  * Designed for easy understanding by beginners

* **Finance Learning Modules**

  * Simplified explanations of key finance concepts
  * Interactive inputs instead of static content

* **Clean & Responsive UI**

  * Mobile-friendly design
  * Focus on clarity and usability

## Project Objective

The goal of Finance Lab is to bridge the gap between **finance theory and real-life financial decision-making**. The platform is built to help students, beginners, and non-finance backgrounds understand:

* Personal budgeting
* Basic taxation
* Financial planning concepts

in a hands-on, experiment-driven way.
